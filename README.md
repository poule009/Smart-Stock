# Smart-Stock

Ce dépôt contient le backend (Laravel) et le frontend (Angular) de l'application Smart-Stock.

## Structure
- `backend/` : Application Laravel
- `frontend/` : Application Angular

## Installation

### Backend
1. Installer les dépendances PHP :
   ```sh
   cd backend
   composer install
   ```
2. Configurer le fichier `.env`.
3. Générer la clé d'application :
   ```sh
   php artisan key:generate
   ```

### Frontend
1. Installer les dépendances Node.js :
   ```sh
   cd frontend
   npm install
   ```

## Utilisation
- Lancez le backend et le frontend selon vos besoins.
